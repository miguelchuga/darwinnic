POS Invoice On Receipt

Why print invoice if the receipt is enough?

An invoice is used to record the details of the merchandise sold. Invoice number is one of the key difference between an Invoice and receipt. It is a unique, sequential code that is systematically assigned to invoices.
If you are an Odoo Point of Sale user and finding it is taking a lot of your time to keep the records then the module named as POS Invoice On Receipt can help you. Now, you can print invoice number in the receipt also which is time as well as effort saving. The key difference between receipt and invoice has overcome with the help of the module.

Salient Features
  • Invoice Number is also available in Receipt.
  • Saves the time of the seller to print an extra invoice.
  • Easy use and Installation.
  • The module enables to keep the record of invoice on the receipt as well
  • No need to print the invoice to know the Invoice Number.
  • The invoice number will be also visible on XmlReceipt.
